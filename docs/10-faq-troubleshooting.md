# 10 FAQ & Troubleshooting

## Q1: What is the recommended way to get started?
A: First run `pip install -e .`, then use `python -m raisex.cli.eval_cli ...`.

## Q2: Where are the datasets located?
A: All datasets are under `data/datasets/...`.

## Q3: Where are the configuration files?
A: All configs are under `configs/search_space`, `configs/algorithms`, and `configs/experiments`.

## Q4: How do I use the Python API?
A: Use `from raisex.api.public import ...`.
